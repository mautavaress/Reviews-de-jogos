**Plataforma jogada**: GameCube (via DolphinUWP)

**Tempo total**: `1h`

**Status**: Incompleto, mas com algumas sidequests concluídas

| Pontos positivos  | Pontos negativos  |
| ------------ | ------------ |
| Pode ser jogado com amigos  |  Ausência de história[^1] |

[^1]: Se comparado com o Mega Microgame$ de GBA.

> [!WARNING]
> Esta análise é apenas primeiras impressões. Pode mudar caso eu jogue por mais tempo!

O jogo está para um remaster do famoso WarioWare de GBA, com foco no multiplayer, embora dê pra jogar solo. A parte ruim é que o jogo não tem história. Só possui fases inspiradas em personagens do WarioWare com os mesmo temas da versão de GBA, mas com trilha sonora pouco diversificada. Portanto, só recomendo este jogo se for jogar com amigos. Pra jogar sozinho, aí é melhor usar o GBA.

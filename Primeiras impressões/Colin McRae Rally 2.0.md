O jogo não é para qualquer um, é pra quem manja mesmo de rally, assim como os DiRTs mais novos. Os controles são bons mas tem que saber fazer as curvas de forma certa. Apesar disso, assim como acontece em Sega Rally, o jogo sinaliza as curvas, alertando previamente o jogador, que terá um espaço de tempo curto para tomar uma decisão (frear, virar, etc.).

**Plataforma jogada**: PlayStation 2 (via XBSX2)

**Tempo de jogo**: `2h52min (in-game)`

**Status**: Incompleto, mas com algumas missões secundárias concluídas

| Critério  | Avaliação (0,5 a 10,0)  |
| ------------ | ------------ |
|  Geral | `7,0` |

| Pontos positivos  | Pontos negativos  |
| ------------ | ------------ |
|   | Controles truncados  |
|   |  Cãmera não pode ser controlada |


A versão original de PS2 mostra o quanto que a série evoluiu. Apresenta controles truncados e a câmera não pode ser controlada - pelo menos, o remake melhora em muitos aspectos. Recomendo jogar as histórias secundárias antes de prosseguir na história, já que a conclusão delas rende dinheiro, itens e EXP, que podem ser usados pra adquirir aprimoramentos, tornando o personagem mais forte, especialmente em lutas mais importantes. Recomendo também jogar a versão “Restored”, feita por fãs, que restaura a dublagem em japonês e corrige a localização ocidental - ou jogar a versão Kiwami mesmo.
